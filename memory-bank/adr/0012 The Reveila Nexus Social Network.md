# The "Nexus" Social Layer

To boost the Enterprise Edition's confidence, we launch a Moltbook-style playground called Reveila Nexus.

What it is: A web-based "Showroom" where users can share their Agent Personalities (e.g., "The Legal Triage Agent") without sharing their private data.

How it works: Users export a "Skill Blueprint" (the logic/prompts/tools) but not the SQLite memory file.

The Viral Loop: "I built a Reveila Agent that handles all my M&A due diligence—here is the Skill link."
